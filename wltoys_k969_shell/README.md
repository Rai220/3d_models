# WLToys K969 — кузов спорткара

Тонкостенный кузов для RC-модели WLToys K969 (масштаб 1:28). Открыт снизу, крепится клипсами на 4 стойки шасси.

## Параметры

- Длина: 175 мм, ширина: 75 мм
- Колёсная база: 98 мм
- Толщина стенки: 1.2 мм (FDM)

## Как это работает

Кузов описан набором поперечных сечений (нижняя часть + кабина). Модуль `seq_hull()` строит плавную оболочку через последовательный hull между соседними сечениями. Колёсные арки и крепёжные отверстия вырезаются из готовой формы.

```bash
openscad -o wltoys_k969_shell.stl body_shell.scad
```
