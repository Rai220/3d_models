# LDRC — кузов хэтчбека

Тонкостенный кузов хэтчбека для RC-модели LDRC (масштаб 1:28). Открыт снизу, крепится клипсами на 4 стойки шасси.

## Параметры

- Длина: 175 мм, ширина: 75 мм
- Колёсная база: 98 мм
- Толщина стенки: 1.2 мм (FDM)

## Отличия от K969

Профиль кабины типа хэтчбек — крыша начинается раньше и резко обрывается сзади (крутой угол заднего стекла).

```bash
openscad -o ldrc_body.stl body_shell.scad
```
