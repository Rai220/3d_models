# Brotherhood of Steel — многоцветное панно

Рельефное настенное панно с логотипом Brotherhood of Steel из Fallout. 5 цветов для AMS/мультиматериальной печати.

<img src="FO3_BoS_logo.webp" width="400" />

## Слои

| # | Деталь | Цвет |
|---|--------|------|
| 1 | Подложка | белый |
| 2 | Крылья | тёмно-синий |
| 3 | Шестерёнки | чёрный |
| 4 | Меч | серебристый |
| 5 | Рамка | тёмно-серый |

## Сборка

```bash
# 1. Сегментация изображения на цветовые слои
python3 generate_layers.py

# 2. Экспорт STL для каждой детали
bash export_stl.sh

# 3. Сборка в один 3MF с назначенными цветами
python3 build_3mf.py
```

## Как это работает

1. **generate_layers.py** — разбивает растровое изображение на цветовые маски (по HSV), конвертирует в SVG через potrace
2. **bos_panel.scad** — импортирует SVG-слои, выдавливает с нужной толщиной, вырезает пересечения между деталями (булевы операции в 2D для скорости)
3. **build_3mf.py** — собирает все STL в один 3MF-файл с материалами и цветами для слайсера
