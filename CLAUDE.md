# 3D Models

Репозиторий для хранения 3D-моделей.

## Инструменты

- **OpenSCAD** — основной инструмент для создания и редактирования моделей. Модели описываются программно на языке OpenSCAD (файлы `.scad`).

## Принципы работы с OpenSCAD

### Производительность булевых операций

- **Булевы операции (difference, intersection, union) выполнять в 2D, а не в 3D.** Сначала объединить/вычесть 2D-профили, затем `linear_extrude`. CGAL boolean на 3D-мешах из SVG может зависать на часы, а в 2D — доли секунды.
- Выделять 2D-профили в отдельные модули (`_2d_wings()`, `_2d_gears()` и т.д.), чтобы переиспользовать их и в 2D-булевых, и в 3D-extrude.

### Многоцветная печать (AMS / MMU)

- Каждая цветовая деталь — отдельный PART. Экспорт через `openscad -D "PART=N" -o part.stl model.scad`.
- Детали **не должны пересекаться** — использовать `difference()` для вырезания перекрытий.
- Приоритет слоёв: верхние детали (меч) вырезаются из нижних (крылья, шестерёнки).

### Сборка 3MF

- Для удобства импорта в слайсер собирать все STL в один `.3mf` файл с назначенными цветами.
- 3MF — это ZIP-архив с XML (`[Content_Types].xml`, `_rels/.rels`, `3D/3dmodel.model`).
- Структура модели: каждая деталь — отдельный `<object>` с `pid/pindex` для привязки к `<basematerials>`, затем один композитный `<object>` с `<component>` для сборки.
- STL из OpenSCAD бывают ASCII — парсер должен поддерживать оба формата.
- При конвертации STL → 3MF дедуплицировать вершины (STL дублирует вершины для каждого треугольника).
