# Mandelbrot — рельефное панно

Настенное панно с фракталом Мандельброта. Рельеф генерируется математически — граница множества образует самую высокую часть.

## Сборка

```bash
# 1. Генерация карты высот
python3 generate_heightmap.py > heightmap.dat

# 2. Экспорт STL
bash export_stl.sh
```

## Детали для многоцветной печати

| # | Деталь | Описание |
|---|--------|----------|
| 1 | Основание | плоская подложка |
| 2 | Рельеф | фрактальная поверхность |
| 3 | Рамка | обрамление панно |

## Как это работает

`generate_heightmap.py` вычисляет smooth iteration count для каждой точки комплексной плоскости и преобразует его в высоту через логарифмическую функцию. OpenSCAD `surface()` читает полученную карту высот и строит 3D-рельеф.
